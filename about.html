О проекте Pixel Tactics

Pixel Tactics — это фанатский сайт, вдохновлённый одноимённой настольной игрой, в которой сочетаются стратегия, тактика и атмосфера 16-битных RPG в современном цифровом исполнении.

Цель проекта

Создать интерактивную платформу, передающую дух оригинальной игры — где игрок может изучить героев, правила и почувствовать эстетику ретро-игр в неоновом дизайне.

Особенности

  • Автоматическая карусель с неоновыми эффектами
  • Анимированные карточки героев с эффектом переворота
  • Интерактивный слайдер правил
  • Полная адаптивность для всех устройств
  • Форма обратной связи с валидацией ввода

Технологии

Проект создан с использованием HTML5, CSS3, JavaScript и jQuery. Для адаптивности применяется Bootstrap Grid, а все анимации реализованы на CSS и jQuery.

Авторы и назначение

Сайт разработан в рамках учебного задания для демонстрации навыков веб-разработки: структурной вёрстки, адаптивного дизайна, интерактивных компонентов и анимации.
