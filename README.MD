## Project Overview

The main goal of the project is to create an informative and interactive website that allows users to:
- Explore available heroes and their abilities;
- Learn the rules of the tactical card game;
- Create their own custom hero;
- Contact the developer and learn more about the project.

The design follows a **retro neon pixel-art aesthetic**, inspired by old-school video games.
